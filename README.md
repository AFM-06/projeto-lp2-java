# Projeto LP2 - Orientação a Objetos
Projeto desenvolvido para a conclusão da cadeira de **Linguagem de Programação II**.

## Sobre o projeto
Para a conclusão de LP2 no período de **2025.2**, foi requisitada a apresentação de um projeto autoral que abordasse de forma criativa os assuntos vistos ao longo do estudo em Orientação a Objetos.

## Desenvolvimento
Minha solução para o projeto foi a criação de um "RPG" simples com 3 personagens (Arqueiro, Mago, Lutador), onde todos herdam comportamentos em comum de uma classe abstrata chamada **Personagem**.

O objetivo principal era estabelecer um PVP justo baseado em rodadas. Para a implementação dos personagens em um ambiente assim, foi criada uma classe **Jogo** que contém todos os comportamentos fundamentais para o funcionamento da partida. Ela gerencia a escolha de personagens pelo jogador 1 e jogador 2, além de controlar o sistema de turnos, onde os jogadores podem escolher como atacar.
